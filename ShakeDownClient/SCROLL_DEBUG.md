# Скролл панели настроек - ГОТОВО ✅

## Что реализовано:

### 1. Плавный скролл с физикой
- **Spring physics**: плавное движение с инерцией
- **Damping**: естественное замедление
- **Target-based**: скролл движется к целевой позиции

### 2. GL11 Scissor
- Используется `org.lwjgl.opengl.GL11` напрямую
- Правильная конвертация координат (bottom-left origin)
- Учитывается scale factor окна

### 3. Увеличенные высоты
- **Aura**: 600px (было 500px)
- **Triggerbot**: 200px (было 150px)
- **Speed**: 150px (было 80px)
- **Removals**: 300px (было 200px)
- **SwingAnimations**: 200px (было 150px)

### 4. Визуальный скроллбар
- Показывается только когда есть что скроллить
- Фиолетовый цвет (в стиле GUI)
- Отражает текущую позицию

## Как использовать:

1. **Открой настройки модуля** (ПКМ по модулю с настройками)
2. **Наведи курсор на панель настроек** (справа от меню)
3. **Крути колесико мыши** - настройки плавно скроллятся
4. **Смотри скроллбар** справа - показывает позицию

## Особенности:

### Плавная анимация
- **Spring constant**: 0.15 (скорость реакции)
- **Damping**: 0.82 (замедление)
- Естественное движение с инерцией

### Автоматическое определение области
- Скролл работает только когда курсор над панелью
- Не мешает скроллу основного контента

### Scissor clipping
- Настройки не выходят за границы панели
- Использует GL11 для точного обрезания

## Текущие значения:

- **Видимая высота**: guiHeight - 100 = 240px (при guiHeight=340)
- **Aura высота**: 600px
- **Максимальный скролл**: 600 - 240 = 360px
- **Скорость скролла**: 30px за один тик колесика

## Параметры физики:

```java
// Spring physics constants
SCROLL_SPRING = 0.15f;  // Скорость реакции
SCROLL_DAMPING = 0.82f; // Замедление

// Scroll speed
verticalAmount * 30;    // 30px за тик колесика
```

## Модули с длинными настройками:

- **Aura**: 600px (требует скролл)
- **Removals**: 300px (требует скролл)
- **Triggerbot**: 200px (может требовать скролл)
- **SwingAnimations**: 200px (может требовать скролл)
- **Speed**: 150px (обычно влезает)
