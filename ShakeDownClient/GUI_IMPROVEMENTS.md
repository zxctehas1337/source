# GUI Improvements - Исправления интерфейса

## Исправленные проблемы

### 1. Сохранение выбранной вкладки
**Проблема:** Пользовательское расположение вкладок не сохранялось между запусками GUI.

**Решение:** 
- Перенесена загрузка последней выбранной вкладки в `SwingUtilities.invokeLater()` для гарантии полной инициализации компонентов
- Теперь вкладка загружается после того, как все компоненты GUI готовы
- Конфигурация `lastSelectedTab` корректно сохраняется и загружается

### 2. Видимость вкладки Visuals
**Проблема:** Вкладка Visual ESP при запуске GUI была частично не видна.

**Решение:**
- Исправлена инициализация вкладок - теперь контент загружается после полной инициализации панелей
- Вкладка "Visuals" теперь корректно отображается с первого запуска

### 3. Компактность интерфейса
**Проблема:** Вкладки занимали слишком много места из-за больших отступов.

**Решение:**
Уменьшены отступы во всех компонентах:

#### MainScreen (главное окно):
- Размер кнопок вкладок: 130x36 → 110x30
- Отступы кнопок: 8,15,8,15 → 6,12,6,12
- Размер шрифта кнопок: 14 → 13
- Отступы между кнопками: 6px → 4px
- Ширина панели вкладок: 130px → 110px
- Отступы панели: 15px → 10px
- Отступы заголовка: 15px → 10px
- Размер шрифта заголовка: 22 → 20
- Размер кнопки закрытия: 18 → 16

#### VisualsTab:
- Отступ после заголовка: 15px → 10px
- Отступы между секциями: 12-15px → 8-10px
- Отступы между чекбоксами: 6px → 4px

#### HudTab:
- Отступ после заголовка: 20px → 10px
- Отступ после кнопки: 20px → 12px
- Отступы в секции Visual Aura: 10px → 6px, 5px → 4px
- Отступ перед секцией: 25px → 15px

#### SettingsTab:
- Отступ после заголовка: 20px → 10px
- Отступ после чекбокса: 20px → 12px
- Отступы между слайдерами: 20px → 10px
- Отступ перед секцией конфигурации: 25px → 15px
- Отступ после секции: 10px → 6px

#### AboutTab:
- Размер шрифта заголовка: 32 → 28
- Отступ после заголовка: 10px → 6px
- Отступ после версии: 30px → 15px
- Отступ после автора: 20px → 12px
- Отступ перед кнопками: 30px → 15px

#### MiscTab:
- Отступ после заголовка: 20px → 10px

## Результат
- GUI стал на ~30% компактнее
- Все вкладки теперь помещаются на экране без прокрутки
- Сохранение выбранной вкладки работает корректно
- Вкладка Visuals отображается правильно с первого запуска
