## Особенности дизайна

✨ **Современный минимализм**
- Чистые линии
- Много пространства
- Фокус на контенте

🎨 **Цветовые акценты**
- Фиолетовый для активных элементов
- Серый для неактивных
- Белый текст для читаемости

🎭 **Плавные анимации**
- Slide-up при открытии (250ms)
- Hover эффекты
- Toggle анимации

📱 **Адаптивный дизайн**
- 2 колонки для модулей
- Автоматический расчет размеров
- Работает на любом разрешении
